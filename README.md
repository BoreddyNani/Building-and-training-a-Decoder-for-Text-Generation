This project investigates the impact of key hyperparameter
choices on the stability, convergence, and performance
of a custom GPT-like decoder-only language model (LM). Using
the IMDB dataset, experiments were conducted comparing
different learning rates (le-2 vs. 1e-3), the presence of gradient
clipping, various optimization algorithms (Adam and AdamW).
The findings highlight the critical role of the learning rate and
optimizer choice in managing training stability and achieving
optimal convergence.
